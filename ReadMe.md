Kompanion.ai is an AI-powered productivity companion that helps users plan tasks, evaluate completion probability, and recover from setbacks through intelligent coaching. Users can create tasks based on priority, estimated duration, and fatigue level, while the system provides productivity insights and task viability analysis.

The platform supports a dual-mode architecture. Non-AWS users can access all core productivity features through a local simulation environment, while AWS users can connect their cloud account to enable Amazon Bedrock-powered AI coaching and Amazon DynamoDB-based task persistence. This approach ensures accessibility while demonstrating practical integration of Generative AI and cloud services.

Tech Stack: Python, Streamlit, HTML/CSS, SVG UI Components, Amazon Bedrock, Amazon DynamoDB, Boto3, Claude 3.5 Sonnet.
